Steam OpenID



Login with Steam and we can automatically generate a profile with recommendations for you. At no time do we have access to your steam credentials or any private information.

Manual Entry

Enter your unique Steam id in order to generate a profile and personalized recommendations. If you are not sure what your Steam id is, then we suggest heading over to Steam ID Finder and tracking down the profile ID (7656119xxxxxxxxxx). With this identifier we are able to see what games own and what games you have been playing so that we can give you personalized game recommendations. You may have to run steamidfinder twice since the first value they give back may look like the following: STEAM_0:1:xxxxxxx. If so, input this value back in and retrieve the profile ID.

This only works if your profile is public, as we do not have access to any private information from your profile
